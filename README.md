# FRAME · AI-анализ смет на ремонт

ML-модуль цифровой платформы **FRAME** (суперапп для управления ремонтом).
Модель оценивает справедливую стоимость ремонтных работ и автоматически
находит в смете подрядчика **подозрительно завышенные или заниженные позиции** —
это фича 5.2 продукта FRAME («Анализ смет»).

> Заявка на **Junior ML Contest ITMO** (AI Talent Hub). Автор: Артемий Вербицкий.

## Задача

- **Регрессия:** предсказать справедливую цену за единицу работы по её параметрам
  (категория, единица измерения, объём, регион, класс ремонта).
- **Детекция аномалий:** сравнить фактическую цену в смете с предсказанной и
  пометить позиции с аномальным отклонением (риск завышения/занижения).

## Данные

У платформы пока нет собственных данных, поэтому датасет **синтезирован на основе
реальных рыночных ориентиров цен на ремонтные работы в РФ (2024–2025)**. Базовые
цены, региональные и «классовые» коэффициенты заданы явно и задокументированы в
[`src/generate_dataset.py`](src/generate_dataset.py) — методология прозрачна и
воспроизводима.

## Как запустить

```bash
python -m venv .venv
.venv\Scripts\activate          # Windows
pip install -r requirements.txt
python src/generate_dataset.py  # сгенерировать датасет
```
